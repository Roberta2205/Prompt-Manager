<h1>PROMPT MANAGER</h1>

<p>
Essa semana eu tive a honra de participar de um projeto incrível realizado pela Rocketseat e pelo Rodrigo Gonçalves. 
Neste projeto foi desenvolvido um Prompt Manager, que nada mais é do que uma aplicação web prática e intuitiva permitindo<br>
o armazenamento de todos os seus prompts de Inteligência Artificial, garantindo mais organização e um acesso rápido no dia a dia.<br>
<br>
Com esse projeto eu aprendi a utilizar a IA do próprio VSCODE e entendi como ela pode ser útil no dia a dia, <br>
garantindo muita agilidade na hora de codar um projeto. Mas veja bem, é sempre recomendado revisar tudo o que a IA<br>
adicionou ao código antes de finalizar, pois nem sempre ela entrega as melhores opções, viu? 
</p>


<h1>Com a aplicação é possível:</h1>

<p>
  <ul>
    <li>Criar e editar os arquivos</li>
    <li>Copiar os prompts para colar direto na IA que você utiliza</li>
    <li>Buscar os que você já crio</li>
  </ul>
</p>
<br>
