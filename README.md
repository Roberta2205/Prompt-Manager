<h1>PROMPT MANAGER</h1>

<p>
Eu tive a honra de participar de um projeto incrível realizado pela Rocketseat e pelo Rodrigo Gonçalves. 
Neste projeto foi desenvolvido um Prompt Manager, que nada mais é do que uma aplicação web prática e intuitiva permitindo
o armazenamento de todos os seus prompts de Inteligência Artificial, garantindo mais organização e um acesso rápido no dia a dia.
Com esse projeto eu aprendi a utilizar a IA do próprio VSCODE e entendi como ela pode ser útil no dia a dia, 
garantindo muita agilidade na hora de codar um projeto. Mas veja bem, é sempre recomendado revisar tudo o que a IA
adicionou ao código antes de finalizar, pois nem sempre ela entrega as melhores opções, viu? 
</p>

<h1>Acesse ao site</h1>
<p>https://nlwpromptmanagerr.netlify.app/</p>


<h1>Funcionalidades</h1>
 
<p>
  <ul>
    <li>Criar e editar os arquivos</li>
    <li>Copiar os prompts para colar direto na IA que você utiliza</li>
    <li>Buscar os que você já crio</li>
  </ul>
</p>

<h1>Como utilizar</h1>
<p>
  <ul>
    <li>Ao clicar no título do prompt você pode adicionar um título para o seu prompt</li>
    <li>Ao clicar no segundo card você adiciona o prompt que gostaria de guardar</li>
    <li>Ao clicar em salvar o seu prompt fica salvo no site permitindo que você acesse ele a qualquer hora</li>
    <li>Em buscar por título você pode pesquisar todos prompts que guardou</li>
    <li>Em novo prompt pode sempre adicionar um prompr novo</li>
    <li>Ao clicar na seta no canto superior esquerdo a aba de novo prompt some e ao clicar novamente ela reaparece</li>
  </ul>
</p>
